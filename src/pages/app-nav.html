Overview

The eui-app-nav component provides a responsive, collapsible side navigation drawer. It handles routing and active state management automatically.

Usage

Place the eui-app-nav component at the root of your application, typically alongside your main content wrapper.

<eui-app-nav></eui-app-nav>

To populate the links, you set the navItems property on the element using JavaScript.

import { icons } from "@3r1s_s/erisui"; const navItems = [ { label: "Home", path: "/", icon: "home" }, { label: "Settings", path: "/settings", icon: "settings" }, { type: "divider" }, { label: "About", path: "/about", icon: "info" } ]; document.querySelector("eui-app-nav").navItems = navItems; Customization

You can customize the width of the navigation drawer using CSS variables.

--nav-width Default: 75px
--nav-expanded-width Default: 250px
