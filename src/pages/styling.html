Overview

ErisUI comes with a pre-built design system that handles colors, typography, and layout. It relies heavily on CSS Variables (Custom Properties), making it extremely easy to customize at runtime or via your own CSS.

Usage

To use the styles, you simply need to include the CSS file in your project options.

NPM

import "@3r1s_s/erisui/style.css";

CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/3r1s-s/erisui-2@dist/erisui.css"> Theme Variables

You can override any of these variables in your own :root block to change the look of the application.

:root { --app-accent: #ff5bff; /* Change global accent color */ --app-text: #ffffff; }
Variable Description
--app-text Main text color.
--app-accent Primary accent color for interactive elements.
--app-100 to --app-900 Background grayscale ramp.
--nav-width Width of the collapsed navigation.

Light Mode

ErisUI supports light mode out of the box. Simply add the .light class to your :root or html element.
