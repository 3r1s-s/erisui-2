Usage

Sliders reflect a range of values along a bar, from which users may select a single value.

<eui-slider value="50"></eui-slider> <eui-slider value="75" min="0" max="100" step="10"></eui-slider> <eui-slider disabled value="30"></eui-slider> Attributes
Attribute Description
value The current value of the slider.
min The minimum value (default: 0).
max The maximum value (default: 100).
step The step interval (default: 1).
disabled Whether the slider is disabled.
no-thumb Hides the slider thumb
vertical Makes the slider vertical
thickness Changes the slider thickness
no-number Hides the slider value
